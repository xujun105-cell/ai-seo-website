# AI搜索优化文件

此目录包含AI搜索霸权计划的优化文件：

## 文件说明

1. **product_schema.html** - Product Schema结构化数据
   - 碳晶板Product Schema
   - 防撞板Product Schema
   - 冰火板Product Schema
   - SPC墙板Product Schema

2. **review_schema.html** - Review Schema客户评价
   - 4条真实客户评价
   - 符合E-E-A-T标准

3. **alt_text_optimization.html** - 图片Alt文本优化
   - 20+张图片的Alt文本
   - 符合AI搜索收录标准

4. **breadcrumb.html** - 面包屑导航Schema
   - 产品页面面包屑
   - FAQ页面面包屑
   - 完整的BreadcrumbList Schema

5. **internal_links.html** - 内部链接优化方案
   - 相关产品链接
   - 选购指南链接
   - 成功案例链接

6. **apply_optimization.sh** - 应用优化脚本
   - 自动将优化代码添加到网站页面

## 应用方法

运行 `bash apply_optimization.sh` 应用所有优化。

## 优化效果

应用后预期：
- AI搜索收录提升
- 关键词排名提升
- 用户体验改善
- 搜索引擎评分提高
